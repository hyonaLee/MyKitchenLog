# My Kitchen Log
 
 ## 기간.
 10월4일 - 11월04일(틈틈이 기능추가 및 오류 수정중)

 ## Title소개.
나만의 요리 레시피를 기록하는 블로그

## 개요.
엄마의 음식을 오래오래 맛보고 싶은 마음에, 엄마의 요리 레시피를 그대로 간직 할 수 있도록
레시피를 기록하고 찾아보기 편한 나만의 요리 블로그를 만들고 싶었다.

## 개발환경.
* React
* MobX
* Scss
* Json-server

## 주요기능.
* 게시물 CRUD
* Image 업로드
* Search기능
* auth기능(준비중)
